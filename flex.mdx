WHAT IS FLEXBOX?

A CSS3 layout that provides an easy and clean way to arrange items within a container.

* No FLOATS!
* Responsive and mobile friendly
* Positioning and child elements is much easier
* Flex container's margins do not collapse with the margins of its contents.
* Order of elements can easily be changed without editing the source HTML


FLEX PROPERTIES

* Display : flex | Inline-flex;
* Flex-direction : row | column
* Flex-wrap : wrap | no wrap | wrapreverse
* Flex - basis : <length>
* Justify-content : flex-start | flex-end |center
* Align-items : flex-start | flex-end | center
* Align-content : flex-start | flex-end | center

flex-grow : <number>
flex-shrink : <number>
flex : <integer>
order : <integer>
